后记

请求处理流程简单描述

  1. 请求http://localhost:8080/simpleDemo/home/index
  2. 被web.xml中配置的org.springframework.web.servlet.DispatcherServlet拦截
  3. 解析请求地址home/index
  4. 通过HandlerMapping找到HomeController类的映射（@RequestMapping(value = "/home")），再找到home方法的映射（@RequestMapping(value = "/index", method = RequestMethod.GET)）
  5. 调用home方法后，根据返回类型（String）和返回值（return "index";），通过spring-servlet.xml中配置的org.springframework.web.servlet.view.InternalResourceViewResolver，在/WEB-INF/views/中，找到index.jsp，渲染视图
  6. 返回给浏览器
