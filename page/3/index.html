The Origin of Linux

发表于 2017-12-24 更新于 2024-11-07 分类于 coder ， Tool Disqus：

Linux 是一种自由和开源的类Unix操作系统。

严格意义来说 Linux 是一种操作系统的内核（Kernel）。因操作系统中包含了许多图形用户界面（GUI）和其他实用工具。如今 Linux 常用来指基于 Linux 的完整操作系统，内核则改以 Linux内核 称之。

说到 Linux 的起源，就不得不说 Unix 。正如其定义：类Unix操作系统。

而说到 Unix 就也不得不说 C语言。
