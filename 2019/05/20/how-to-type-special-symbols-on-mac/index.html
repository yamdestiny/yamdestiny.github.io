如何在 Mac 上输入特殊符号 

发表于 2019-05-20 更新于 2024-11-07 分类于 coder ， Tool Disqus：

快速输入特殊符号

字符检视器

在有光标的位置，可以打开字符检视器，可以快速输入包括表情（😊）、标点（❣）、货币（£）等符号。

  • 按下 Control + Command + 空格键
  • 点击右上角状态栏中的输入法图标，选择 显示表情与符号

打开窗口后，点击右上角窗口右上角的 ⌘ 图标，可以展开窗口显示更多。

点击左上角的 ⚙︎ ，选择自定列表，可以选择更多。

输入法

使用中文输入法，在有光标的位置，可以打开一个快速输入中文常用字符和颜文字的选词框。

  • 按下 Shift + Option + B
  • 点击右上角状态栏中的输入法图标，选择 显示文字表情与符号

快捷键

  • Apple()：Shift + Option + K
  • 美分(￠)：Option + 4
  • 英镑(￡)：Option + 3
  • 日元/人民币(￥)：Option + Y
  • 欧元(€)：Shift + Option + 2
  • 度(°)：Shift + Option + 8
  • 约等于(≈)：Option + X
  • 除号(÷)：Option + /
  • 无穷(∞)：Option + 5
  • 小于等于(≤)：Option + ,
  • 大于等于(≥)：Option + .
  • 不等于(≠)：Option + =

使用不同的输入法，可能会有不同。如记不住，可以通过打开输入法的虚拟键盘（点击右上角状态栏中的输入法图标，选择 显示虚拟键盘 ），可以快速查看，按住键，虚拟键盘会自动变化显示的按键图标。

LaunchBar

使用 LaunchBar 替换系统自带的 Spotlight（聚焦搜索），可以快速输入字符
