E-Book List

发表于 2019-04-15 更新于 2024-11-07 分类于 memories Disqus：

收集的在线电子书

EN Book List

  • HTTP (HyperText Transfer Protocol)

    HTTP 协议基础知识

  • Test-Driven Web Development with Python

    介绍如何以 TDD 的方式开发互联网应用，使用 Django 框架举例

  • Algorithms

    开源英文电子书，作者是美国伊利诺伊大学的教师，本书是他的讲课笔记，偏重于数学

  • Statistics Done Wrong

    统计会犯错

  • Lectures · hacker tools

  • PC Assembly Book

  • Introduction to Theoretical Computer Science

  • 汇编语言教程

    汇编语言

  • Acceptance Test Driven Development with React

    前端测试

  • Site Reliability Engineering

    网站可靠性工作手册

  • Introduction to Computer Organization with x86-64 Assembly Language & GNU/Linux

    计算机系统导论：以汇编语言与 Linux 系统为例

  • HACKING WITH SWIFT ONLINE

  • Ops School Curriculum

    运维工程师的培训资料库，介绍各种运维的基础知识。

  • Computer Networks: A Systems Approach

    • Source

    计算机网络：系统方法

  • Computing Before Computers

    计算机之前的计算设备，这是一本免费电子书，介绍计算机出现之前的人类计算设备，从算盘、差分机、穿孔计算机，直到电子计算器。

  • How Does a Database Work?

    如何构建一个简单的数据库，介绍如何用 C 语言实现一个类似 sqlite 的数据库

  • Simply Logical: Intelligent Reasoning by Example

    通过实例介绍如何通过 Prolog 语言实现智能推理。

  • Forecasting: Principles and Practice

    预测：原理与实践。如何通过数据进行预测。

  • Introduction to Computer Organization: ARM Assembly Language Using the Raspberry Pi

    计算机组织简介：以 ARM 汇编语言和 Raspberry Pi 为例

  • How to create an OS from scratch

    如何从头开始创造一个操作系统

  • Blockchain Technology Overview

    区块链技术概述。美国国家标准技术研究所（NIST）编写

  • Computer Networking: Principles, Protocols and Practice

    计算机网络：原理，协议和实践。解释了计算机网络的主要原理以及互联网的关键协议，用于高年级本科网络课程。

  • An Intensive Introduction to Cryptography

    加密学导论。介绍加密背后的数学知识，哈佛大学的本科生教材。

  • Practical Common Lisp

  • TensorFlow-Course

  • Foundations of Databases

    数据库基础

  • CodementorIO / Python-Learning-Resources

    Python 学习资源

  • LINUX 就该这么学

  • The Linux Command Line (TLCL) CN

  • 任正非思想

  • Python - 100天从新手到大师
